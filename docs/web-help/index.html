音色とボイシング

項目 選択肢 内容
音色 Piano / Pad / Organ / Pluck 試聴と再生プレビューに反映
ベース なし / 1 / 2 / 両方 1オクターブ下、2オクターブ下、または両方のベース音を追加
開き なし / 3度 / 5度 第3音または第5音を1オクターブ上へ移してオープンボイシング化
