# Chord Draft 利用ガイド

## 概要
Chord Draft は、コード進行をテキスト入力またはテンプレートから組み立て、試聴し、MIDI として書き出すための軽量デスクトップアプリです。

主な特徴:
- コード名、ディグリーネーム、アラビア数字で入力可能
- タイムライン上でコードの並べ替え、長さ変更、転回形変更が可能
- テンプレートの適用、編集、別名保存、上書き保存
- MIDI 書き出しと D&D エクスポート
- ピアノ、パッド、オルガン、プラックの試聴音色

## 基本操作
### 入力から進行を作る
1. 左の `入力` 欄へコード進行を入力します。
2. `Ctrl+Enter` で解析してタイムラインへ反映します。
3. タイムライン上のコードをクリックすると試聴できます。

### 保存と書き出し
- `保存`: プロジェクトを JSON で保存
- `MIDI`: MIDI ファイルを書き出し
- `D&D`: 一時 MIDI を生成して DAW へドラッグ

## 入力方式
### 1. コード名入力
例:
```text
C G Am F
F Em7b5 A7 | Dm7 G7
C/E F G7 Bm7b5
```

対応例:
- メジャー / マイナー: `C`, `Am`
- 7th 系: `G7`, `Cmaj7`, `Dm7`
- ハーフディミニッシュ: `Bm7b5`
- 分数コード: `C/E`, `B/D#`
- 拡張和音: `add9`, `sus2`, `sus4`, `aug`, `dim`, `9`

### 2. ディグリーネーム入力
例:
```text
I V vi IV
ii7 V7 Imaj7
Im7 IV V7
viiø7 III7 vi7
```

対応表記:
- ローマ数字小文字: `vi`, `ii7`
- 大文字 + m: `Im`, `IIm7`
- 和音変化: `Imaj7`, `Vsus4`, `viiø7`

### 3. アラビア数字入力
現在の `キー / 調性` を基準に解釈します。

例:
```text
1 5 6 4
2m7 5svn 1mas
7hdm 3 6m
```

基本:
- `1` から `7`: ダイアトニックコード

短縮コマンド:
- `m`: minor
- `M`: major
- `hdm` または `m75`: half diminished
- `six` または `6`: sixth
- `svn` または `7`: seventh
- `mis` または `m7`: minor seventh
- `mas` または `M7`: major seventh
- `adn` または `add9`: add9
- `sut` または `sus2`: sus2
- `suf` または `sus4`: sus4

例:
- `6m` -> 6度のマイナー
- `5svn` -> 5度の 7th
- `1mas` -> 1度の maj7
- `7hdm` -> 7度の m7-5

## 長さ指定
### 1. 通常入力
単純入力では `既定長` が適用されます。

例:
```text
I V vi IV
```

### 2. 小節線と小節内分割
`|` で小節を区切ると、小節内のコード数から長さを推定します。

例:
```text
F | Em7b5 A7 | Dm7 G7
```

この場合:
- `F` は 1小節
- `Em7b5` と `A7` は各 2拍
- `Dm7` と `G7` も各 2拍

### 3. ハイフンによる保持
`-` は「前のコードをそのスロットぶん保持する」記号です。

例:
```text
G#sus4 - - G# - - - -
```

`スロット` 設定が 8 のとき、同じ小節を 8 分割して長さを解釈します。

### 4. 明示長さ指定
変則長さは `(n/8)` で指定できます。

例:
```text
F(3/8) Em7b5(5/8)
B(3/8) G#m(5/8)
```

対応例:
- `3/8`
- `5/8`
- `7/8`

## 2段目の設定
### 拍子
- `4/4`
- `3/4`

### スロット
テキスト入力時の小節内分解単位です。
- `4`
- `8`

`-` を使った入力で効きます。

### 既定長
小節指定がない通常入力の初期長さです。

### 優先表示
- `ディグリー`
- `コード`

タイムラインや候補の主表示を切り替えます。

### 音色
- `Piano`
- `Pad`
- `Organ`
- `Pluck`

試聴と再生プレビューに反映されます。

## タイムライン操作
### 選択
- コードカードをクリックすると選択 + 試聴

### 並べ替え
- ドラッグで移動
- `Ctrl+←` / `Ctrl+→` でも移動

### 長さ変更
- カード内の長さメニュー
- `Alt+←` / `Alt+→`

### 転回形変更
- 2段目の `転回`
- `Alt+↑` / `Alt+↓`

### 付点
- 2段目の `付点`
- `Alt+.` でも切替

### 選択コードのその他操作
2段目の `操作` から実行:
- `複製`
- `削除`
- `←`
- `→`

## 和音操作
2段目の `和音操作` を開くと、選択コードに対して和音変換を適用できます。

対応:
- `解除`
- `7`
- `min7`
- `maj7`
- `sus2`
- `sus4`
- `6`
- `aug`
- `dim`
- `m7-5`
- `add9`
- `9`

適用時は音が鳴ります。

## テンプレート
### 一覧
- テンプレート選択
- ダブルクリックで即適用
- `適用` ボタンで反映

### フェイルセーフ設計
一覧で選んだだけでは編集欄に流れません。

編集は明示的に開始します。
- `編集`: 選択テンプレートをドラフトへコピー
- `新規作成`: 空ドラフト

### エディタ
- `入力読込`: 現在の入力欄をドラフトへ
- `進行読込`: 現在のタイムラインをドラフトへ
- `別名保存`: 新規テンプレートとして保存
- `上書き保存`: 編集中テンプレートへ保存
- `削除`: 編集中テンプレートを削除

### テンプレート欄のショートカット
- `進行` テキストエリアは `Ctrl+Enter` で確定

## ショートカット一覧
### 全体
- `Ctrl+N`: 新規
- `Ctrl+O`: 開く
- `Ctrl+S`: 保存
- `Ctrl+E`: MIDI 書き出し
- `Ctrl+Z`: アンドゥ
- `Ctrl+Y`: リドゥ
- `Ctrl+Shift+Z`: リドゥ

### 再生
- `Space`: 再生 / 停止

### タイムライン
- `← / →`: 前後コードを選択して試聴
- `Delete` / `Backspace`: 選択コード削除
- `Ctrl+← / Ctrl+→`: 選択コード移動
- `Alt+← / Alt+→`: 長さ変更
- `Alt+↑ / Alt+↓`: 転回形変更
- `Alt+.`: 付点切替

### 入力欄
- `Ctrl+Enter`: 解析してタイムライン反映

### テキストボックス一般
- `Enter`: 確定してフォーカスを外す

## スケール判定について
コード名入力時は、内容から `キー / 調性` を推定します。

ただし相対調で曖昧な場合があります。
- 例: `Bメジャー` と `G#マイナー`

そのため、曖昧な進行では:
1. 一度自動判定させる
2. 必要なら `キー / 調性` を手で修正する

の使い方を推奨します。

## 現状の注意点
- 音色はアプリ内プレビュー用です。MIDI 自体の音色指定ではありません
- 変則長さは `(n/8)` 指定が最も安全です
- `-` は前コード保持記号として解釈されます
- `%` と `=` は直前コードの繰り返しとして解釈されます

## 推奨入力パターン
### 王道進行
```text
1 5 6m 4
```

### ジャズ寄り
```text
I viiø7 III7 | vi7 II7
```

### 変則拍
```text
F(3/8) Em7b5(5/8)
```

### 譜面寄りの保持
```text
G#sus4 - - G# - - - - | G#sus2 - - G# - - - -
```
