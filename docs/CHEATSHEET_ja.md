# Chord Draft チートシート

## 最短入力
```text
1 5 6 4
```

## 小節付き
```text
F | Em7b5 A7 | Dm7 G7
```

## 保持
```text
G#sus4 - - G# - - - -
```

## 繰り返し
```text
B | % | G#m | %
E | = | B | =
```

## 変則長
```text
F(3/8) Em7b5(5/8)
```

## 数字入力短縮
```text
1M 2mis 5svn 7hdm 4sut 4suf 6adn
```

## ショートカット
```text
Ctrl+Enter 反映
Ctrl+N      新規
Ctrl+O      開く
Ctrl+S      保存
Ctrl+E      MIDI
Ctrl+Z      Undo
Ctrl+Y      Redo
Space       再生/停止
Ctrl+←→     並べ替え
Alt+←→      長さ
Alt+↑↓      転回
Alt+.       付点
```
