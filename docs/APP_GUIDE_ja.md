# Chord Draft アプリ内ガイド

## できること
- コード名、ディグリー、アラビア数字で進行入力
- タイムライン上で試聴、並べ替え、長さ変更
- テンプレートの適用、編集、別名保存、上書き保存
- MIDI 書き出しと D&D エクスポート
- 音色、ベース追加、オープンボイシングの切り替え

## 上段
- 左: プロジェクト名、新規、開く、保存
- 中央: スケール、BPM、再生、停止、ループ
- 右: アンドゥ、リドゥ、MIDI、D&D

## 2段目
- 拍子
- スロット
- 既定長
- 優先表示
- 音色
- ベース
- 開き
- 操作
- 転回
- 付点
- 和音操作
- 移調

## 入力欄
- コード進行を入力
- `Ctrl+Enter` で反映
- `%` と `=` は直前コードの繰り返し
- `-` は前コード保持
- `|` は小節区切り
- `(3/8)` のような明示長も使える

## タイムライン
- クリック: 選択 + 試聴
- ドラッグ: 並べ替え
- 長さプルダウン: 個別長さ変更

## 音色
- `Piano`
- `Pad`
- `Organ`
- `Pluck`

## ベース
- `なし`
- `1`
- `2`
- `両方`

現在は以下の意味で実装しています。
- `1`: 1オクターブ下のベース追加
- `2`: 2オクターブ下のベース追加
- `両方`: 1オクターブ下と 2オクターブ下を両方追加

## 開き
- `なし`
- `3度`
- `5度`

現在は以下の意味で実装しています。
- `3度`: 第3音を1オクターブ上へ
- `5度`: 第5音を1オクターブ上へ

## テンプレート
- 一覧選択では編集されません
- `適用`: タイムラインへ反映
- `編集`: 選択テンプレートをドラフトへコピー
- `新規作成`: 空ドラフト
- `入力読込`: 入力欄をドラフトへ
- `進行読込`: タイムラインをドラフトへ
- `別名保存`: 新規保存
- `上書き保存`: 編集中テンプレートへ保存

## 和音操作

- `maj7`
- `sus2`
- `sus4`
- `6`
- `aug`
- `dim`
- `m7-5`
- `add9`
- `9`

## ショートカット
- `Ctrl+N`: 新規
- `Ctrl+O`: 開く
- `Ctrl+S`: 保存
- `Ctrl+E`: MIDI 書き出し
- `Ctrl+Z`: 元に戻す
- `Ctrl+Y`: やり直す
- `Ctrl+Shift+Z`: やり直す
- `Space`: 再生 / 停止
- `Ctrl+← / Ctrl+→`: 選択コード移動
- `Alt+← / Alt+→`: 長さ変更
- `Alt+↑ / Alt+↓`: 転回変更
- `Alt+.`: 付点切替
- `Delete` / `Backspace`: 選択コード削除

## 数字入力
単独数字:
- `1 5 6 4`

短縮コマンド:
- `m`: min
- `M`: maj
- `hdm` / `m75`
- `six` / `6`
- `svn` / `7`
- `mis` / `m7`
- `mas` / `M7`
- `adn` / `add9`
- `sut` / `sus2`
- `suf` / `sus4`
